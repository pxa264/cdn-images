# 分类图片目录

## 说明

分类图片用分类的slug命名：`{slug}.jpg`

## 规范

- **尺寸**: 400x300px
- **格式**: JPG
- **大小**: < 200KB

## CDN访问格式

```
https://cdn.jsdelivr.net/gh/pxa264/cdn-images@main/categories/{slug}.jpg
```

## 示例

```
categories/
├── electronics.jpg     # 数码家电
├── clothing.jpg        # 时尚美妆
├── home.jpg           # 家居百货
└── books.jpg          # 图书办公
```
