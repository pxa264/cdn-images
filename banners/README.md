# Banner轮播图目录

## 说明

用于首页轮播的Banner图片

## 规范

- **尺寸**: 1200x400px
- **格式**: JPG
- **大小**: < 1MB

## CDN访问格式

```
https://cdn.jsdelivr.net/gh/pxa264/cdn-images@main/banners/{filename}.jpg
```

## 示例

```
banners/
├── home-banner-1.jpg   # 首页轮播图1
├── home-banner-2.jpg   # 首页轮播图2
└── promo-banner.jpg    # 促销活动Banner
```
